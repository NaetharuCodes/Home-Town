# This doc explains how building in the game works.

In-game the build system requires that the player has a character contact an architect (you can do so via phone, or later in game by computer or smart phone app). This sends out an NPC to visit. While they are on site, you (the player) can then plan your build. This is just a blueprint. You can adjust it as much as you like, and decide where things will go.

Once you have that done you lock it in. Note that the Architect does not do the design at all. They simply function as an in game event that then lets the player do the design.

Once locked in the build is set for a future time (say 2 to 3 days in advance). On that morning a truck with builders turns up, and they carry out the work. They return each day until the work is completed. Visually they set up a small building site, have things like tools and a cement mixer etc. This gives the player a strong feeling that the event is happening in-game. At the end the player is left with a brand new house / extension / conversion of their property.

They can also do the same for re-decorations. Adding new carpet, paint etc can be done by the player characters as an activity (with skill based outcomes) or you can hire in handy men to do it professionally for a price. Likewise with installing appliances, doing plumbing, electrical work etc.
